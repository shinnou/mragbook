本章で解説したプログラムを動かすための実行手順を示しておきます。ライブラリのインストールは全て済んでいることを仮定しているので、不足のライブラリなどがあれば、その都度、インストールして下さい。

(1) 4章で利用した data ディレクトリをまるごとコピーする（リンクを張ってもよい）

(2) IP アドレスを持つ Ubuntu マシンに ollama のサーバーを起動させる

(3) プログラム内の関数 ollama.Client を呼び出す以下の箇所で、host の IP アドレス部分を (2) のサーバーの IP アドレスに変更する

(4) RAG を実行、クエリはプログラム内で設定

```bash
$ pip install langgraph
$ python mma_rag.py
```
