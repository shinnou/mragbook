本章で解説したプログラムを動かすための実行手順を示しておきます。ライブラリのインストールは全て済んでいることを仮定しているので、不足のライブラリなどがあれば、その都度、インストールして下さい。

---

(1) OpenAPI の API Key の設定

本書ではこの件は説明していませんが、本章のプログラムを動かすには必要です（他の章のプログラムでは不要です）。概略述べれば、まず OpenAI 開発者プラットフォームでキーを発行し、それをプログラムの環境変数（OPENAI\_API\_KEY）として PC に保存するか、コード内の初期化設定に直接記述します。セキュリティのためにキーを直接書き込まず、.env ファイルなどの環境変数から読み込む形式にするのが一般的です。本書のプログラムでは API Key は環境変数に保存されていることを前提にしていますので、この設定部分がプログラムには記述されていません。

---

(2) data ディレクトリを作成し、前章で作成した ibaraki.txt.gz を解凍して、ibaraki.txt をその下に置く

---

(3) データベースを構築する

```bash
$ pip install openai
$ python create_db.py
```

> data/ibaraki_vec_db.json が作成される

---

(4) RAG を実行、クエリはプログラム内で設定

```bash
$ python generate_ans.py
```

> *** 回答が表示される ***
